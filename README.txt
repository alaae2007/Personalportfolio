Personal Portfolio Website
Created by Alaa Elammar

This website is a personal portfolio made using HTML, CSS and JavaScript.

It includes:
- a floating navigation bar
- an about me section
- a projects section
- an animated skills section
- a contact form
- responsive design for desktop, tablet and mobile

Files included:
- index.html
- images folder
- robots.txt
- sitemap.xml
- README.txt

This website was created to demonstrate front-end web development skills and knowledge of HTML, CSS, JavaScript, bot-readable files and human-readable text files.
